# rokuTestProject


Pre-Requisites
1. Roku device in developer mode. Instructions linked [here](https://developer.roku.com/en-ca/docs/developer-program/getting-started/developer-setup.md).
2. Install provided HelloWorld.zip in project to roku device. Follow in structions on sideloading channel in previous link.
3. Download Roku Development Project located [here](https://github.com/rokudev/automated-channel-testing).
4. Run RokuWebServer found in src folder (main.go)
5. In Config.java, update ROKU_IP to ip address of roku device.

Run Tests in Test Folder!
